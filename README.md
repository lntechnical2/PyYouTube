# PyYouTube 


Get Video Data from YouTube link 

## Installation 
```bash
pip install py-youtube
```

## How to use it ?
#### Get Videos Data 

```python
from py_youtube import Data
data = Data("https://youtu.be/HhHzCfrqsoE").data()
print(data)
```
